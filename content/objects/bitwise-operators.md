
The bitwise operators expect integer input (and will truncate float values).

| operator           | description            |
|--------------------|------------------------|
| {{< pdobj "&&" >}}    | logical `and`          |
| {{< pdobj "||" >}}    | logical `or`           |
| {{< pdobj "&" >}}     | bitwise `and`          |
| {{< pdobj "|" >}}     | bitwise `or`           |
| {{< pdobj ">>" >}}    | right signed bit shift |
| {{< pdobj "<<" >}}    | left signed bit shift  |
