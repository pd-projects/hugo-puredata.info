### [wrap]

The “wrap” object wraps the input to a value between 0 and 1, including negative numbers (for instance, -0.2 maps to 0.8.).

INLET:

- float - input value to given function.

OUTLET:

- float - the result of the operation.

ARGUMENT:

- NONE.
